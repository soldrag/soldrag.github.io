# Personal Website

[![Build Status](https://img.shields.io/github/actions/workflow/status/soldrag/soldrag.github.io/node.js.yml?branch=main)](https://github.com/soldrag/soldrag.github.io/actions)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](./LICENSE)

A personal portfolio website built with [Next.js](https://nextjs.org/), [React](https://react.dev/), [TypeScript](https://www.typescriptlang.org/), and [Tailwind CSS](https://tailwindcss.com/).

**[See it live →](https://soldrag.github.io)**

## Get Started

### Local Development

```bash
git clone https://github.com/soldrag/soldrag.github.io.git
cd soldrag.github.io
npm install
npm run dev
```

Requires Node.js 20+ ([nvm](https://github.com/nvm-sh/nvm) recommended).

### GitHub Codespaces

1. Click **Code** → **Codespaces** → **Create codespace**
2. Run `npm run dev`

No local setup needed. Everything runs in your browser.

## Commands

```bash
npm run dev      # Start dev server
npm run build    # Build for production
npm run format   # Format code
npm test         # Run tests
```

## Deploy

Push to `main` and GitHub Pages deploys automatically.
